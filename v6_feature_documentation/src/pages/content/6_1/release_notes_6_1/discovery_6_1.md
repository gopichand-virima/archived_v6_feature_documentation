# Discovery

## New Features

### Enhanced Scan Engine Performance

Improved the core discovery scan engine to support faster network traversal and parallel device polling, reducing overall scan times for large environments.

### Credential Vault Integration

Added support for external credential vault systems, allowing administrators to securely manage and rotate scan credentials without storing them directly in Virima.

## Enhancements

### Scan Scheduling Improvements

Enhanced the scan scheduling interface to support more granular time-based controls, including maintenance window awareness and blackout period configuration.

### Network Range Management

Improved the network range configuration interface with bulk import/export capabilities and validation for overlapping subnets.

## Bug Fixes

### Scan Status Reporting

Fixed an issue where scan progress indicators could display incorrect completion percentages for large subnet scans.

### Credential Rotation

Resolved a timing issue that could cause scan failures during automated credential rotation events.
