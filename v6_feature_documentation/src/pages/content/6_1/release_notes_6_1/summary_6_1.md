# Summary

**Release Date:** 2026

**Version:** 6.1

---

Virima 6.1 delivers significant enhancements across discovery, CMDB, ITSM, and reporting modules. This release focuses on improved automation, expanded integration capabilities, and enhanced user experience throughout the platform.

---

## Highlights

| Module | Key Changes |
|--------|-------------|
| Discovery | Enhanced scan engine performance and new credential management |
| CMDB | Improved relationship mapping and CI lifecycle management |
| ITSM & ITAM | Streamlined ticket workflows and asset tracking enhancements |
| Integration | New REST API endpoints and webhook support |
| Admin | Role-based access control improvements and audit logging |
| Reports | New dashboard widgets and export format options |
| MSP | Multi-tenant management enhancements |
| Beta | Preview features available for early feedback |
| Discovery Application | Application dependency mapping improvements |
