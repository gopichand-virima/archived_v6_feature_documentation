# Beta

## New Features

### AI-Assisted Search (Preview)

Introducing an AI-powered search capability that understands natural language queries and returns contextually relevant results from across the Virima platform. This feature is available as a preview and may change before general availability.

### Predictive Analytics Dashboard (Preview)

Added a predictive analytics dashboard that uses historical data patterns to forecast asset lifecycle events, capacity trends, and potential service disruptions. Available for early adopter feedback.

## Enhancements

### Beta Feature Toggle

Enhanced the beta feature management interface to allow administrators to enable or disable individual preview features per user role, providing granular control over early access.

## Known Issues

### AI Search Accuracy

The AI-assisted search is trained on the current documentation corpus and may return less relevant results for highly specialized or recently added content areas. Accuracy improvements are ongoing.

### Dashboard Data Latency

The predictive analytics dashboard may experience a delay of up to 15 minutes when processing historical data for large environments. Optimization is planned for a future update.
