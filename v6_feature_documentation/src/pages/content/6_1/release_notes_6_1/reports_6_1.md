# Reports

## New Features

### Custom Dashboard Widgets

Added the ability to create custom dashboard widgets with configurable data sources, visualization types (charts, tables, KPIs), and refresh intervals.

### Scheduled Report Delivery

Introduced automated report scheduling with support for recurring delivery via email in multiple formats (PDF, Excel, CSV) to configured recipient lists.

## Enhancements

### Export Options

Enhanced report export capabilities with additional format options, improved formatting fidelity for PDF exports, and support for large dataset pagination in Excel exports.

### Filter Persistence

Improved report filtering with saved filter profiles, cross-report filter sharing, and user-level default filter configurations.

## Bug Fixes

### Chart Rendering

Fixed an issue where pie charts with more than ten data segments could render with overlapping labels in PDF export format.

### Date Range Filters

Resolved a timezone handling issue that caused date range filters to exclude the last day of the selected range in certain regional configurations.
